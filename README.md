N11 Bootcamp Homework Repository

This repository contains my solutions for the N11 Bootcamp assignments.

All homework solutions are organized by weeks inside the src directory.

src

      ├── week03homework

Each folder represents a weekly assignment and contains its own solution.

The purpose of this repository is to practice:

Object-Oriented Programming (OOP)
SOLID principles
Clean code structure in Java
Basic software design concepts
